A little about me

As a passionate game developer with a strong foundation in Unity and C#, I specialize in creating engaging gameplay systems and robust developer tools. With a keen eye for detail from my background in photography and 3D art, I strive to build immersive and polished experiences. I am constantly exploring new technologies to push the boundaries of interactive entertainment.

What I Do

Game Development

Building core mechanics, AI behavior, and UI systems in Unity/C# to create engaging and interactive experiences.

Tools Development

Creating custom editor tools, plugins, and standalone utilities to improve and accelerate the development workflow.

Prototyping

Rapidly developing and iterating on new game ideas and concepts to test for fun and viability.

Engaging Gameplay Systems

I focus on writing clean, efficient, and scalable C# code to build core gameplay mechanics. From character controllers and AI behavior to complex game state management, I am dedicated to creating a seamless and enjoyable player experience.


Efficient Tools & Workflow

A great game is built on a great workflow. I specialize in developing custom tools and editor extensions for the Unity IDE that empower designers and artists to work more effectively, reducing iteration time and improving overall quality.


Interactive 3D Experiences

Leveraging my skills in 3D art and Unity's powerful rendering pipeline, I work on bringing game worlds to life. This includes shader programming, lighting, particle effects, and performance optimization to ensure a smooth and visually compelling result.
